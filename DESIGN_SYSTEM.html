Dark Mode Card Guidelines

Key Principles:

  • Use var(--pivot-gray-900) for card backgrounds
  • Apply subtle purple glow on hover: 0 0 0 1px rgba(139, 92, 246, 0.1)
  • Increase shadow opacity for better depth perception
  • Consider glassmorphism for special cards
  • Ensure AA contrast ratio for all text

Example Implementation:

/* Dark Mode Card */ [data-theme="dark"] .card { background: var(--pivot-gray-900); border-color: var(--pivot-gray-800); box-shadow: 0 1px 3px 0 rgb(0 0 0 / 0.3); } [data-theme="dark"] .card:hover { border-color: rgba(139, 92, 246, 0.3); box-shadow: 0 4px 6px -1px rgb(0 0 0 / 0.3), 0 0 0 1px rgba(139, 92, 246, 0.1); } /* Glassmorphism Effect */ [data-theme="dark"] .glass-card { background: rgba(30, 41, 59, 0.5); backdrop-filter: blur(10px); border: 1px solid rgba(139, 92, 246, 0.2); }
